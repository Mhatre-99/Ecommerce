This is a simple web app which compares products from two websites Amazon and Flipkart and displays the best product.
The web app only considers product from the first page of the result given by both the websites.
Products not meeting the criterion for comparison are not considered, such as if a product's rating is not available.
It also creates an excel file where the user can see all the products considered for comparison from both the websites.
I have used flask, bootstrap, css and html .
